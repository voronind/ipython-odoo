Project description

Install
-------
```commandline
pip install ipython_odoo
```

Development
-----------
We need installed `pyenv` and `pipenv`.
```console
git clone git@github.com:voronind/ipython-odoo.git

cd ipython-odoo
pipenv install --dev
```

Run tests:
```console
pipenv run fulltest
```
